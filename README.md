# Deskripsi Aplikasi
Aplikasi Janji Temu Dokter Rumah Sakit merupakan aplikasi berbasis Java Swing yang dirancang untuk mengelola data janji temu antara pasien dan dokter berdasarkan departemen medis. Aplikasi ini menyediakan fitur CRUD (Create, Read, Update, Delete) untuk pengelolaan data pasien, pemilihan departemen, dokter, serta jadwal pemeriksaan melalui antarmuka GUI.

Dalam pengembangannya, aplikasi menerapkan konsep Object-Oriented Programming (OOP) dengan penggunaan class, object, serta struktur data seperti HashMap dan ArrayList guna mendukung pengelolaan data yang terorganisir. Sistem ini juga dilengkapi dengan mekanisme login sederhana sebagai autentikasi awal, sehingga memberikan gambaran implementasi sistem informasi rumah sakit sederhana yang interaktif dan fungsional.
